* 端口：可在面板上查看
* 密码：可在面板上查看
* 加密方式：aes-256-cfb

## 计费细节
每消耗 100M 流量触发一次扣费(约 0.06 元), 若扣费导致帐号余额低于 0, 则会自动关闭 ShadowSocks 服务。当手动关闭 ShadowSocks 时, 强行触发一次扣费，不足 100M 按 100M 计算。
